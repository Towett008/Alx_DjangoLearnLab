\# Django Admin Setup for Book Model



1\. Registered the Book model in `bookshelf/admin.py`

2\. Created a superuser with `python manage.py createsuperuser`

3\. Logged in at `http://127.0.0.1:8000/admin/`

4\. Verified that the Book model appears with:

&nbsp;  - Columns: title, author, publication\_year

&nbsp;  - Filter by publication\_year

&nbsp;  - Search box for title and author
